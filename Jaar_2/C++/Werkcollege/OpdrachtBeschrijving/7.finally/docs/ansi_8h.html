Detailed Description

Defines various ansi color-code strings.

See e.g. ansi escape code
Available are:
AS_xxx (set style to: BOLD, UNDERLINE or INVERTED)
AC_xxx (set foreground color to: BLACK, RED, GREEN, BROWN, BLUE, MAGENTA, CYAN or WHITE)
AB_xxx (set background color to: BLACK, RED, GREEN, BROWN, BLUE, MAGENTA, CYAN or WHITE)
AA_RESET (reset to normal)

Note
If the ECLIPSE pre-processor token was defined the strings will be left empty because the eclipse console will not handle ansi color-codes.
If this file is used on a non-unix platform the strings will be left empty because it is unlikely that that platform (e.g. windows) will handle ansi codes.
