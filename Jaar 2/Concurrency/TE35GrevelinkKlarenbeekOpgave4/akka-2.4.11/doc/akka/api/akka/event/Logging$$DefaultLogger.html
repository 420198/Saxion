akka.event.Logging

DefaultLogger

Related Doc: package Logging

class DefaultLogger extends Actor with StdOutLogger with RequiresMessageQueue[LoggerMessageQueueSemantics]

Actor wrapper around the standard output logger. If akka.loggers is not set, it defaults to just this logger.
